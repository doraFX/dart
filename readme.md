## 这是什么？

**照片飞镖盘（Photo Dartboard）** 是一款：

- 🕹️ **纯前端 Web JS 游戏**
- 🎯 **真实飞镖物理手感**
- 🖼️ **支持上传任意照片作为飞镖盘**
- 📱 **手机 / 桌面通用**
- 🧠 **操作极简，非常解压**

你只需要：

1. 上传一张照片  
2. 长按屏幕 / 鼠标开始蓄力  
3. 上下移动进行瞄准  
4. 松手 —— **射出飞镖**

---

## 特性一览

- 🎯 **真实飞镖抛物线**
  - 重力
  - 速度
  - 命中误差随难度变化
- 🧠 **三档难度**
  - 距离更远
  - 抖动更大
  - 手感更诚实
- 📐 **可调飞镖盘尺寸**
- 🖼️ **任意图片作为靶面**
- 🔊 **Web Audio 原生音效**
  - 蓄力音
  - 命中音
  - Bull 特殊反馈
- 🎥 **Three.js 真 3D**
  - 非假 2.5D
  - 非 Canvas 糊图
